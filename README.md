## 📖 Overview

**URL Shortener Chrome Extension** is a browser-based tool that allows users to quickly shorten any URL directly from their browser — eliminating the need to open separate URL shortening websites.

Built with simplicity and speed in mind, this extension enhances productivity and streamlines link sharing.

---

## ✨ Features

- 🔗 **Instant URL Shortening** — Generate short links in one click
- ⚡ **No Website Visit Needed** — Works directly inside Chrome
- 📋 **Copy to Clipboard** — Easily copy shortened URLs
- 🧩 **Chrome Extension Integration** — Seamless browser experience
- 🎯 **Minimal UI** — Clean and distraction-free interface

## 🚀 How It Works

1. Open any webpage
2. Click on the extension icon
3. Click **"Shorten URL"**
4. Copy and share your shortened link instantly
